## 📂 Installation

Clone the repository

```bash
git clone https://github.com/ganesh-j-m/ganesh-more.git
```

Go to the project folder

```bash
cd ganesh-more
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
